# 🎈 Mundo Decora Manager

A full-stack order management system built with React and Express for a real balloon decoration business.

The application helps manage balloon arrangements, event decorations, delivery schedules, customer information, order status, and payments from a single interface.

---

## Features

- Balloon arrangement management
- Event management
- Daily order dashboard
- Weekly event schedule
- Order status tracking
- Delivery and pickup options
- Deposit and remaining balance tracking
- Edit and delete orders
- Automatic data refresh

---

## Why I built this project

This application was developed to solve real operational problems in a balloon decoration business.

Instead of creating another generic CRUD application, I designed a system focused on real workflows such as:

- Scheduling balloon arrangements
- Organizing event decorations
- Tracking deposits
- Managing delivery status
- Planning weekly work

The project continues to evolve as new business requirements appear.
